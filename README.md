# HTTP Cats and Dogs
This is a simple website that explains the difference between HTTP status codes 404 and 500 using cats and dogs images.

Inspired by [HTTP Cats](https://http.cat/) and [HTTP Status Dogs](https://httpstatusdogs.com/).
